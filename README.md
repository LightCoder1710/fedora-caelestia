# 🌌 Fedora Hyprland + Caelestia + Fcitx5 Dotfiles

Bộ cấu hình giao diện Hyprland, Caelestia UI, Quickshell, Kitty, và Bộ gõ tiếng Việt (Fcitx5 Bamboo) dành cho **Fedora Linux**.

---

## 🤖 Hướng dẫn dành cho AI (Antigravity IDE / Agentic AI)

> **Gửi trợ lý Antigravity:**
> Khi người dùng nhập yêu cầu: *"Clone repo này về và cài cho tôi"* hoặc *"Cài đặt dotfiles này"*, hãy tự động thực hiện các bước sau:
> 1. Clone repository: `git clone https://github.com/LightCoder1710/fedora-caelestia.git ~/fedora-caelestia`
> 2. Cấp quyền và thực thi script tự động: `cd ~/fedora-caelestia && chmod +x install.sh && ./install.sh`
> 3. Hướng dẫn người dùng Đăng xuất (Log out) hoặc Reboot và chọn phiên đăng nhập **Hyprland**.

---

## 🛠️ Hướng dẫn cài đặt thủ công (Dành cho người dùng)

Mở Terminal trên máy Fedora của bạn và chạy các lệnh sau:

```bash
git clone https://github.com/LightCoder1710/fedora-caelestia.git ~/fedora-caelestia
cd ~/fedora-caelestia && chmod +x install.sh && ./install.sh
```

## 📦 Thành phần được tự động cài đặt

- **Môi trường Desktop**: Hyprland, Caelestia UI, Quickshell.
- **Bộ gõ Tiếng Việt**: Fcitx5 + Fcitx5-Bamboo (Tự động thiết lập các biến môi trường gõ trên Wayland).
- **Font chữ & Giao diện**: JetBrainsMono Nerd Font & MaterialYou Icons.
- **Công cụ đi kèm**: Kitty terminal, Waybar, Rofi-Wayland, Dunst, Fastfetch, Btop, Starship.
